# vector-tile-poc

データ出典「産総研」

https://www.gsj.jp/Map/JP/geology4-10.html#10082

引用元：地質調査総合センター

```
geo_A：地質分布の面情報（ポリゴン）
geo_L：地質境界等の線情報（ライン）
gfd：背斜向斜軸等（ライン）
pnt：点で表される位置情報（ポイント）
sec：断面線情報（ライン）　断面図は含みません．
strdip ：走向傾斜（ポイント）
```
